```bash
# 1. 复制环境变量文件
cp .env.example .env

# 2. 编辑 .env 文件
nano .env

# 3. 安装依赖
go mod download

# 4. 启动服务
go run main.go

# 5. 测试 API
curl http://localhost:8080/health
```
